# Little Lemon Restaurant 🍋
A modern Mediterranean restaurant website built with React, offering a seamless digital dining experience.

## About The Project
Little Lemon is a modern Mediterranean restaurant specializing in fresh ingredients and authentic flavors. This project is a comprehensive restaurant website developed as a capstone project for a web development course, showcasing modern frontend development practices and responsive design principles.
Live Demo: Coming Soon - GitHub Pages deployment
Features

### 🍽️ Interactive Menu Display - Browse through our carefully crafted Mediterranean dishes
### 📅 Table Reservation System - Easy-to-use booking system for restaurant tables
### 📋 Reservation Management - View and manage existing reservations
### ⭐ Customer Reviews - Read testimonials from satisfied customers
### 🏪 About Section - Learn about Little Lemon's story and values
### 📱 Fully Responsive Design - Optimized for desktop, tablet, and mobile devices
### 🎨 Modern UI/UX - Clean, intuitive interface following current design trends

## Built With
### React - Frontend JavaScript library
### HTML5 - Markup language
### CSS3 - Styling and responsive design
### JavaScript (ES6+) - Modern JavaScript features
### React Router - For navigation between pages
### React Hooks - For state management
